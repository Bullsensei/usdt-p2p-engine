# USDT P2P Comparison Engine

P2P USDT price comparison and decision engine for Vietnamese market.

## Project Structure

- `/backend` - Node.js + Express API
- `/frontend` - React web application

## Local Development

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Environment Variables

### Backend

- `PORT` - Server port (default: 3001)

### Frontend

- `REACT_APP_API_URL` - Backend API URL

## Deployment

- Backend: Vercel
- Frontend: Vercel

See deployment guide in `/docs` folder.
